# Projet 4 : Débugger une application Java

## Objectifs
Repérer et corriger les erreurs de conception d'un programme java et optimiser son fonctionnement.

## Description
Programme d'analyse des tendances permettant de lire les symptômes depuis un fichier de données, compter les occurrences et pourcentage d'occurrence de chaque symptôme et écrire les résultats d’analyse dans un fichier de sortie. 

## Fichiers E/S
Les fichiers "symptest.txt" et "Results.out.txt" se situent à cette adresse:
```
 \Users\NameUser\Project_DA_Java_EN_Come_to_the_Rescue_of_a_Java_Application\
```
